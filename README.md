# CS4414 Labs

Rust programming labs for CS4414. Clone this repository and work inside each lab's crate directory.

## Labs

| Lab | Crate | Topic |
|-----|-------|-------|
| 1 | `lab1-counter-stats` | Structs, enums, methods (coming soon) |
| 2 | `lab2-process-registry` | Ownership, encapsulation, heap, borrowing |
| 3 | `lab3-process-registry` | Errors, traits, generics, iterators, HashMap |

## Quick start

```bash
git clone https://github.com/HypexJuice/CS4414-Labs.git
cd CS4414-Labs/lab2-process-registry
cargo test
```

Read `ASSIGNMENT.md` inside each lab crate for the full specification.

## Requirements

- [rustup](https://rustup.rs/) (stable toolchain)
