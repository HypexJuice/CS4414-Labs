Lab 3: Process Registry API
============================

INTRODUCTION
------------

Upgrade the process registry from Lab 2:

  Lab 2                          Lab 3
  Vec<Box<Process>>              HashMap<u32, Box<Process>> keyed by PID
  get(index)                     get_by_pid(pid)
  RegistryError::Full only       Full, DuplicatePid, NotFound
  no removal                     remove_by_pid returns owned Process

The starter src/lib.rs includes Process from Lab 2. Implement the HashMap
registry, Named, iterators, and remove_by_pid.

Put all code in src/lib.rs. Run cargo test when you are done.


SETUP
-----

1. Install rustup: https://rustup.rs/
2. Clone and enter:

   git clone https://github.com/HypexJuice/CS4414-Labs.git
   cd CS4414-Labs/lab3-process-registry

3. Run cargo test. The starter compiles; tests fail until the registry API works.


PART 1 — Process (provided)
---------------------------

Starter includes ProcessState, Process, ProcessError, new, accessors, set_state
(same API as Lab 2). Read that code before starting the registry.


PART 2 — Registry
-----------------

Field processes: HashMap<u32, Box<Process>>. No parallel Vec.

RegistryError: Full, DuplicatePid(u32), NotFound(u32).
MAX_PROCESSES = 8.

new, len, register, get_by_pid — see ASSIGNMENT.md table.
remove_by_pid -> Result<Process, RegistryError>.


PART 3 — Named trait
--------------------

Trait with name(&self) -> &str. Implement for Process.
Starter has a stub impl with todo!() — replace it in Milestone 2.


PART 4 — Iterators
------------------

iter_by_state(state) -> impl Iterator<Item = &Process>
running_pids() -> Vec<u32> via iter_by_state, map, collect.


WORKED EXAMPLE
--------------

reg.register(Process::new(42, "shell".into())?)?;
reg.get_by_pid(42)?.name() == "shell"
let mut init = reg.remove_by_pid(1)?;
init.set_state(ProcessState::Stopped);
reg.running_pids() == [42]


RULES
-----

HashMap sole store. Fallible methods return Result.
iter_by_state uses impl Trait, not dyn Iterator.
No dyn Trait, Rc, Arc, async, unsafe in graded code.
