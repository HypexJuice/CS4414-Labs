Lab 2: Process Registry
=======================

INTRODUCTION
------------

Lab 1 summarized CPU idle samples. Now track the processes on the machine.

Implement a process table. Each process has a PID, name, and state:
Running, Sleeping, or Stopped. Store processes in a Registry backed by
Vec<Box<Process>>.

Lab 1 used public struct fields. Here the fields are private; use methods to
read and update them.

Put all code in src/lib.rs. Run cargo test when you are done.


SETUP
-----

1. Install rustup (stable toolchain): https://rustup.rs/
2. Clone the course labs repository and enter this crate:

   git clone https://github.com/HypexJuice/CS4414-Labs.git
   cd CS4414-Labs/lab2-process-registry

3. Run cargo test. The starter code will not compile until you add the types
   below.


PART 1 — ProcessState AND Process
---------------------------------

ProcessState variants: Running, Sleeping, Stopped.
Derive Copy, Clone, Debug, PartialEq, Eq.

Process private fields: pid (u32), name (String), state (ProcessState).

ProcessError: variant EmptyName.

new(pid, name: String) -> Result<Process, ProcessError>
  Empty name -> Err(EmptyName). Otherwise Ok with state Running.

Accessors on &self: pid -> u32, name -> &str, state -> ProcessState.

set_state on &mut self updates state.


PART 2 — Registry
-----------------

Private field processes: Vec<Box<Process>>.

RegistryError: variant Full.

MAX_PROCESSES: const usize = 8.

new() -> empty Vec.
len(&self) -> count.
register(&mut self, process: Process) -> Result<(), RegistryError>
  At capacity -> Full. Else push Box and Ok.

get(index) -> Option<&Process> on &self.
get_mut(index) -> Option<&mut Process> on &mut self.

Look up by index only. Lab 3 adds PID lookup with HashMap.


WORKED EXAMPLE
--------------

let mut registry = Registry::new();
let init = Process::new(1, "init".into())?;
let shell = Process::new(42, "shell".into())?;
registry.register(init)?;
registry.register(shell)?;
registry.len() == 3
registry.get(1).unwrap().name() == "shell"
registry.get_mut(2).unwrap().set_state(ProcessState::Stopped);


RULES
-----

1. Validation on Process; storage on Registry.
2. register takes Process by value into Vec<Box<Process>>.
3. Allowed: Lab 1 set + Vec, String, Option, Result, Box, const.
4. Do not use: Rc, Arc, HashMap, traits, I/O, unsafe, threads.
